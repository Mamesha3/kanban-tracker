# Kanban Tracker App

A simple Kanban-style task management app to organize work using columns like To Do, In Progress, and Done.

## Features
- Create and manage tasks
- Move tasks between columns
- Kanban-style workflow
- Clean and minimal UI

## What I Learned
- Managing application state with JavaScript
- DOM manipulation and dynamic updates
- Handling user interactions and events
- Organizing UI logic for better readability

## Technologies Used
- HTML
- CSS
- JavaScript
